INSTALLATION:
=============

*ENGLISH*
Grant write privileges for /system and webcounter1.png.
Run install.php to install the WebCounter.
Run index.php for an example how to use WebCounter.

*DEUTSCH*
Gewähre Schreibrechte für den Ordner /system und webcounter1.png.
Starte install.php im Browser um WebCounter zu installieren.
Starte index.php für ein Beispiel, wie WebCounter eingesetzt wird.
